DataJoint

High-level relational database interface for MATLAB

Welcome to DataJoint

DataJoint for MATLAB is a high-level programming interface for relational databases designed to support data processing chains in science labs. DataJoint is built on the foundation of the relational data model and prescribes a consistent method for organizing, populating, and querying data.

DataJoint primarily targets MATLAB but a Python interface is currently in development. MySQL is used as the backend for data storage.

Get started

  1. Clone the DataJoint for MATLAB repository
  2. Download our fork of mym, the MySQL connector for MATLAB, originally developed by Yannick Maret
  3. Follow our tutorial

Who are we?

DataJoint was initially developed in 2009 by Dimitri Yatsenko (Baylor College of Medicine) for the distributed processing and management of large volumes of data streaming from regular experiments. Starting in 2011, DataJoint has been available as an open-source project adopted by other labs and substantially improved with contributions from Andreas Hoenselaar (CalTech) and Alex Ecker (Max Planck Institute).
